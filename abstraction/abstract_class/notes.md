Abstract Class---------
C++ mein Abstract Class ek aisi class hoti hai jiska aap direct object (instance) create nahi kar sakte. Ise as a "blueprint" (design) use kiya jata hai doosri derived classes ke liye.

Kisi bhi class ko abstract banane ke liye, usme kam se kam ek Pure Virtual Function hona zaroori hai.

Pure virtual function ko = 0 laga kar define kiya jata hai, iska class mein koi implementation nahi hota. Derived classes ko ise implement karna hi padta hai.
