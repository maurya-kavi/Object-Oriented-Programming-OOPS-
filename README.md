# Object-Oriented Programming (OOP) in C++

A comprehensive collection of C++ programs demonstrating the fundamental concepts of **Object-Oriented Programming (OOP)**. This repository is designed to help students, beginners, and interview aspirants understand the core principles of OOP through clean, well-structured, and executable examples.

---

## 📚 About

This repository serves as a practical learning resource for mastering Object-Oriented Programming in C++. Each concept is implemented with simple examples and accompanied by comments explaining the underlying logic and behavior.

Whether you're preparing for interviews, learning C++, or revising OOP concepts, this repository provides an organized reference covering the most important topics.

---

## ✨ Features

- Well-structured and beginner-friendly code
- Simple and easy-to-understand implementations
- Covers both theoretical concepts and practical examples
- Suitable for interview preparation
- Written in modern C++

## ⚙️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/maurya-kavi/Object-Oriented-Programming-OOPS-.git
```

```bash
cd Object-Oriented-Programming-OOPS-
```

---

### 2. Compile Any Program

Example:

```bash
g++ access_specifier.cpp -o access_specifier
```

---

### 3. Run the Program

#### Windows

```bash
.\access_specifier.exe
```

#### Linux / macOS

```bash
./access_specifier
```
